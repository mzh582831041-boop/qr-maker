# QR Maker Free

Free online QR code generator. Static site, deploys to GitHub Pages.

## Deploy

1. `git init && git add . && git commit -m "Initial commit"`
2. Create repo `qr-maker` on GitHub
3. `git remote add origin <your-repo-url>`  
4. `git push -u origin main`
5. GitHub repo → Settings → Pages → Source: main branch → Save
